# Policy Di Utilizzo Del Brand SGG / SunGear Games

Versione 1.0 - 2 giugno 2026

Questo documento descrive regole semplici per usare nomi e asset di brand SGG / SunGear Games.

Questo repository e concesso in licenza secondo [LICENSE.md](../../LICENSE.md), con testo russo autorevole della licenza in [LICENSE.ru.md](../../LICENSE.ru.md). La licenza puo consentire l'uso del codice, del pacchetto o di altri materiali del repository secondo i suoi termini, ma non concede il permesso di usare il brand SGG / SunGear Games come nome o branding dei tuoi progetti, prodotti, pacchetti, fork, organizzazioni, domini, siti web o pubblicazioni.

Questa policy non impedisce riferimenti fattuali accurati a SGG / SunGear Games per origine, compatibilita, crediti o attribuzione richiesta. La regola principale e semplice: non usare il brand SGG / SunGear Games come tuo brand e non implicare connessione, supporto o approvazione ufficiale.

## Nomi Protetti

Non usare i nomi seguenti, o varianti confondibilmente simili, nei tuoi nomi, branding, domini, organizzazioni, repository, pacchetti, listing marketplace o build pubbliche:

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

Questo include anche varianti simili con maiuscole diverse, spaziatura, trattini, punti, underscore, traslitterazione, grafia cirillica o errori intenzionali se possono creare confusione con SGG / SunGear Games.

Per esempio, evita varianti come `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames` o `SGG-Games` quando usate nel contesto di software, strumenti, giochi o asset in un modo che puo sembrare abbreviazione, divisione, prodotto ufficiale o brand correlato di SunGear Games.

Questa policy non rivendica diritti esclusivi su parole generiche o descrittive prese da sole. Per esempio, il nome `PerfMeter` senza `SGG`, `SunGear Games`, `SunGearGames` o branding simile non e oggetto di questa policy.

## Regola Generale

Non nominare progetti, prodotti, pacchetti, fork, build, organizzazioni, siti web, domini o pubblicazioni in modo che possa far pensare agli utenti che siano ufficialmente connessi a, rilasciati da, mantenuti da o approvati da SGG / SunGear Games.

## Non Consentito

Senza permesso separato, non usare nomi SGG / SunGear Games:

- nel nome del tuo progetto, prodotto o pacchetto;
- nel nome di un'organizzazione, repository o account GitHub;
- nel nome di un pacchetto Unity, pubblicazione Asset Store o altro listing marketplace;
- in un dominio, sottodominio, sito web o landing page;
- in logo, icona, splash screen, branding UI o materiale marketing;
- nel nome di una build modificata, fork o versione ripacchettizzata.

Esempi di nomi da evitare:

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

Questo elenco non e esaustivo. La regola semplice e: non usare `SGG`, `SunGear Games`, `SunGearGames` o varianti confondibilmente simili come parte del tuo brand.

## Uso Fattuale Consentito

Puoi usare i nomi SGG / SunGear Games per attribuzione fattuale accurata, documentazione o conformita alla licenza.

Per esempio:

> Basato su software di SunGear Games.

Oppure:

> Questo progetto usa codice da un repository SunGear Games.

Questo uso deve essere descrittivo. Non deve apparire come nome del tuo prodotto e non deve implicare connessione, supporto o approvazione ufficiale.

## Fork GitHub Regolari

I fork GitHub regolari usati per pull request, code review, apprendimento o esperimenti personali non devono essere rinominati dentro l'interfaccia fork di GitHub.

Tuttavia, se pubblichi release, build, pacchetti, siti web, documentazione, listing marketplace o altre versioni pubbliche, non usare SGG / SunGear Games come nome o branding di quella versione.

## Loghi E Asset Di Brand

Loghi, icone, immagini, screenshot brandizzati, branding pacchetto e altri asset di brand SGG / SunGear Games non possono essere usati in progetti, prodotti, pubblicazioni, build o materiali marketing di terze parti senza permesso separato.

## Domande

Se non sei sicuro che il tuo uso dei nomi o degli asset di brand SGG / SunGear Games sia consentito, contatta prima l'autore del progetto o apri una issue.
