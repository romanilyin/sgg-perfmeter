# SGG / SunGear Games Brand Usage Policy

Version 1.0 - June 2, 2026

この文書は、SGG / SunGear Games の名称と brand assets を使用するための簡単な規則を説明します。

このリポジトリは [LICENSE.md](../../LICENSE.md) の下でライセンスされており、正本のロシア語ライセンステキストは [LICENSE.ru.md](../../LICENSE.ru.md) です。ライセンスは、その条件に従って code、package、その他 repository materials の使用を許可する場合がありますが、SGG / SunGear Games brand を自身の projects、products、packages、forks、organizations、domains、websites、publications の名称や branding として使用する許可は与えません。

この policy は、origin、compatibility、credits、required attribution のために SGG / SunGear Games へ正確に事実として言及することを妨げません。基本規則は単純です。SGG / SunGear Games brand を自身の brand として使用せず、official connection、support、endorsement を示唆しないでください。

## Protected Names

次の names、または混同を招くほど類似した variants を、自身の names、branding、domains、organizations、repositories、packages、marketplace listings、public builds に使用しないでください。

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

これには、大文字小文字、spacing、hyphens、dots、underscores、transliteration、Cyrillic spelling、intentional misspellings が異なる類似 variants も含まれます。それらが SGG / SunGear Games と混同を生む可能性がある場合です。

たとえば、software、tools、games、assets の文脈で SunGear Games の略称、division、official product、related brand のように見える使い方をする場合、`SunGear`、`Sun Gear`、`Sungear`、`Sun-Gear`、`SGGames`、`SGGGames`、`SGG-Games` などの variants は避けてください。

この policy は、generic または descriptive words 自体について exclusive rights を主張するものではありません。たとえば、`SGG`、`SunGear Games`、`SunGearGames`、または類似 branding を含まない `PerfMeter` という名称は、この policy の対象ではありません。

## General Rule

users が SGG / SunGear Games と officially connected、released by、maintained by、endorsed by であると考える可能性のある形で、projects、products、packages、forks、builds、organizations、websites、domains、publications に名前を付けないでください。

## Not Allowed

別途許可がない限り、SGG / SunGear Games names を次の用途に使用しないでください。

- 自身の project、product、package の名称。
- GitHub organization、repository、account の名称。
- Unity package、Asset Store publication、その他 marketplace listing の名称。
- domain、subdomain、website、landing page。
- logo、icon、splash screen、UI branding、marketing material。
- modified build、fork、repackaged version の名称。

避けるべき names の例です。

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

この list は網羅的ではありません。単純な規則は、`SGG`、`SunGear Games`、`SunGearGames`、または混同を招くほど類似した variants を自身の brand の一部として使用しないことです。

## Allowed Factual Use

正確な factual attribution、documentation、license compliance のために SGG / SunGear Games names を使用できます。

例:

> Based on software by SunGear Games.

または:

> This project uses code from a SunGear Games repository.

このような使用は descriptive である必要があります。自身の product の名称のように見えてはならず、official connection、support、endorsement を示唆してはなりません。

## Regular GitHub Forks

pull requests、code review、learning、personal experiments に使う通常の GitHub forks は、GitHub の fork interface 内で rename する必要はありません。

ただし、自身の releases、builds、packages、websites、documentation、marketplace listings、その他 public versions を公開する場合、その version の name または branding として SGG / SunGear Games を使用しないでください。

## Logos And Brand Assets

SGG / SunGear Games logos、icons、images、branded screenshots、package branding、その他 brand assets は、別途許可がない限り third-party projects、products、publications、builds、marketing materials で使用できません。

## Questions

SGG / SunGear Games names または brand assets の使用が許可されるか不明な場合は、先に project author に連絡するか issue を開いてください。
