# SGG / SunGear Games Brand Usage Policy

Version 1.0 - June 2, 2026

This document describes simple rules for using SGG / SunGear Games names and brand assets.

This repository is licensed under [LICENSE.md](../../LICENSE.md), with the authoritative Russian license text in [LICENSE.ru.md](../../LICENSE.ru.md). The license may allow use of the code, package, or other repository materials under its terms, but it does not grant permission to use the SGG / SunGear Games brand as the name or branding of your own projects, products, packages, forks, organizations, domains, websites, or publications.

This policy does not prevent accurate factual references to SGG / SunGear Games for origin, compatibility, credits, or required attribution. The main rule is simple: do not use the SGG / SunGear Games brand as your own brand, and do not imply official connection, support, or endorsement.

## Protected Names

Please do not use the following names, or confusingly similar variants, in your own names, branding, domains, organizations, repositories, packages, marketplace listings, or public builds:

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

This also includes similar variants with different capitalization, spacing, hyphens, dots, underscores, transliteration, Cyrillic spelling, or intentional misspellings if they may create confusion with SGG / SunGear Games.

For example, please avoid variants such as `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames`, or `SGG-Games` when used in the context of software, tools, games, or assets in a way that may look like a shorthand, division, official product, or related brand of SunGear Games.

This policy does not claim exclusive rights to generic or descriptive words by themselves. For example, the name `PerfMeter` without `SGG`, `SunGear Games`, `SunGearGames`, or similar branding is not the subject of this policy.

## General Rule

Do not name your projects, products, packages, forks, builds, organizations, websites, domains, or publications in a way that may make users think they are officially connected to, released by, maintained by, or endorsed by SGG / SunGear Games.

## Not Allowed

Without separate permission, do not use SGG / SunGear Games names:

- in the name of your own project, product, or package;
- in the name of a GitHub organization, repository, or account;
- in the name of a Unity package, Asset Store publication, or other marketplace listing;
- in a domain, subdomain, website, or landing page;
- in a logo, icon, splash screen, UI branding, or marketing material;
- in the name of a modified build, fork, or repackaged version.

Examples of names to avoid:

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

This list is not exhaustive. The simple rule is: do not use `SGG`, `SunGear Games`, `SunGearGames`, or confusingly similar variants as part of your own brand.

## Allowed Factual Use

You may use SGG / SunGear Games names for accurate factual attribution, documentation, or license compliance.

For example:

> Based on software by SunGear Games.

Or:

> This project uses code from a SunGear Games repository.

Such use must be descriptive. It must not look like the name of your own product and must not imply official connection, support, or endorsement.

## Regular GitHub Forks

Regular GitHub forks used for pull requests, code review, learning, or personal experiments do not need to be renamed inside GitHub's fork interface.

However, if you publish your own releases, builds, packages, websites, documentation, marketplace listings, or other public versions, do not use SGG / SunGear Games as the name or branding of that version.

## Logos And Brand Assets

SGG / SunGear Games logos, icons, images, branded screenshots, package branding, and other brand assets may not be used in third-party projects, products, publications, builds, or marketing materials without separate permission.

## Questions

If you are unsure whether your use of SGG / SunGear Games names or brand assets is allowed, please contact the project author or open an issue first.
