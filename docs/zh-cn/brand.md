# SGG / SunGear Games 品牌使用政策

Version 1.0 - June 2, 2026

本文档说明使用 SGG / SunGear Games 名称和品牌资产的简单规则。

此仓库依据 [LICENSE.md](../../LICENSE.md) 授权，权威俄文许可证文本为 [LICENSE.ru.md](../../LICENSE.ru.md)。许可证可能允许你在其条款下使用代码、package 或其他仓库材料，但不授予你将 SGG / SunGear Games 品牌用作你自己的 projects、products、packages、forks、organizations、domains、websites 或 publications 的名称或品牌的权限。

本政策不阻止为来源、兼容性、credits 或必要 attribution 而准确事实性引用 SGG / SunGear Games。主要规则很简单：不要将 SGG / SunGear Games 品牌作为你自己的品牌，也不要暗示官方联系、支持或认可。

## 受保护名称

请不要在你自己的 names、branding、domains、organizations、repositories、packages、marketplace listings 或 public builds 中使用以下名称或可能造成混淆的相似变体：

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

这也包括大小写、空格、连字符、点、下划线、转写、西里尔拼写或有意拼写错误不同的相似变体，只要它们可能与 SGG / SunGear Games 产生混淆。

例如，请避免在软件、工具、游戏或 assets 语境中使用 `SunGear`、`Sun Gear`、`Sungear`、`Sun-Gear`、`SGGames`、`SGGGames` 或 `SGG-Games` 等变体，因为它们可能看起来像 SunGear Games 的简称、部门、官方产品或相关品牌。

本政策不主张对通用或描述性词语本身拥有排他权利。例如，不带 `SGG`、`SunGear Games`、`SunGearGames` 或相似品牌的 `PerfMeter` 名称不属于本政策对象。

## 一般规则

不要以可能让用户认为你的 projects、products、packages、forks、builds、organizations、websites、domains 或 publications 与 SGG / SunGear Games 存在官方联系、由其发布、维护或认可的方式命名。

## 不允许

未经单独许可，不要在以下位置使用 SGG / SunGear Games 名称：

- 你自己的 project、product 或 package 名称；
- GitHub organization、repository 或 account 名称；
- Unity package、Asset Store publication 或其他 marketplace listing 名称；
- domain、subdomain、website 或 landing page；
- logo、icon、splash screen、UI branding 或 marketing material；
- modified build、fork 或 repackaged version 的名称。

应避免的名称示例：

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

此列表并非详尽。简单规则是：不要将 `SGG`、`SunGear Games`、`SunGearGames` 或容易混淆的相似变体用作你自己品牌的一部分。

## 允许的事实性使用

你可以为了准确的事实性 attribution、documentation 或 license compliance 使用 SGG / SunGear Games 名称。

例如：

> Based on software by SunGear Games.

或者：

> This project uses code from a SunGear Games repository.

此类使用必须是描述性的。它不得看起来像你自己产品的名称，也不得暗示官方联系、支持或认可。

## 常规 GitHub Forks

用于 pull requests、code review、learning 或 personal experiments 的常规 GitHub forks，不需要在 GitHub 的 fork interface 内重命名。

但是，如果你发布自己的 releases、builds、packages、websites、documentation、marketplace listings 或其他 public versions，不要将 SGG / SunGear Games 用作该版本的名称或品牌。

## Logos And Brand Assets

未经单独许可，不得在 third-party projects、products、publications、builds 或 marketing materials 中使用 SGG / SunGear Games logos、icons、images、branded screenshots、package branding 和其他 brand assets。

## 问题

如果你不确定自己对 SGG / SunGear Games 名称或品牌资产的使用是否被允许，请先联系项目作者或 open an issue。
