# Politica De Uso Da Marca SGG / SunGear Games

Versao 1.0 - 2 de junho de 2026

Este documento descreve regras simples para usar nomes e assets de marca SGG / SunGear Games.

Este repositorio e licenciado sob [LICENSE.md](../../LICENSE.md), com o texto russo autoritativo da licenca em [LICENSE.ru.md](../../LICENSE.ru.md). A licenca pode permitir o uso do codigo, pacote ou outros materiais do repositorio sob seus termos, mas ela nao concede permissao para usar a marca SGG / SunGear Games como nome ou branding dos seus proprios projetos, produtos, pacotes, forks, organizacoes, dominios, sites ou publicacoes.

Esta politica nao impede referencias factuais precisas a SGG / SunGear Games para origem, compatibilidade, creditos ou atribuicao exigida. A regra principal e simples: nao use a marca SGG / SunGear Games como sua propria marca, e nao sugira conexao oficial, suporte ou endosso.

## Nomes Protegidos

Por favor, nao use os seguintes nomes, nem variantes confusamente semelhantes, nos seus proprios nomes, branding, dominios, organizacoes, repositorios, pacotes, listings de marketplace ou builds publicas:

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

Isso tambem inclui variantes semelhantes com capitalizacao, espacamento, hifens, pontos, underscores, transliteracao, grafia em cirilico ou erros ortograficos intencionais diferentes se elas puderem criar confusao com SGG / SunGear Games.

Por exemplo, evite variantes como `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames` ou `SGG-Games` quando usadas no contexto de software, ferramentas, jogos ou assets de uma forma que possa parecer abreviacao, divisao, produto oficial ou marca relacionada de SunGear Games.

Esta politica nao reivindica direitos exclusivos sobre palavras genericas ou descritivas isoladas. Por exemplo, o nome `PerfMeter` sem `SGG`, `SunGear Games`, `SunGearGames` ou branding semelhante nao e objeto desta politica.

## Regra Geral

Nao nomeie seus projetos, produtos, pacotes, forks, builds, organizacoes, sites, dominios ou publicacoes de forma que possa levar usuarios a pensar que eles sao oficialmente conectados, lancados, mantidos ou endossados por SGG / SunGear Games.

## Nao Permitido

Sem permissao separada, nao use nomes SGG / SunGear Games:

- no nome do seu proprio projeto, produto ou pacote;
- no nome de uma organizacao, repositorio ou conta do GitHub;
- no nome de um pacote Unity, publicacao na Asset Store ou outro listing de marketplace;
- em dominio, subdominio, site ou landing page;
- em logo, icone, splash screen, branding de UI ou material de marketing;
- no nome de uma build modificada, fork ou versao reempacotada.

Exemplos de nomes a evitar:

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

Esta lista nao e exaustiva. A regra simples e: nao use `SGG`, `SunGear Games`, `SunGearGames` ou variantes confusamente semelhantes como parte da sua propria marca.

## Uso Factual Permitido

Voce pode usar nomes SGG / SunGear Games para atribuicao factual precisa, documentacao ou conformidade com licenca.

Por exemplo:

> Based on software by SunGear Games.

Ou:

> This project uses code from a SunGear Games repository.

Esse uso deve ser descritivo. Ele nao deve parecer o nome do seu proprio produto e nao deve sugerir conexao oficial, suporte ou endosso.

## Forks Regulares Do GitHub

Forks regulares do GitHub usados para pull requests, revisao de codigo, aprendizado ou experimentos pessoais nao precisam ser renomeados dentro da interface de fork do GitHub.

No entanto, se voce publicar suas proprias releases, builds, pacotes, sites, documentacao, listings de marketplace ou outras versoes publicas, nao use SGG / SunGear Games como nome ou branding dessa versao.

## Logos E Assets De Marca

Logos, icones, imagens, screenshots com marca, branding de pacote e outros assets de marca SGG / SunGear Games nao podem ser usados em projetos, produtos, publicacoes, builds ou materiais de marketing de terceiros sem permissao separada.

## Perguntas

Se voce nao tiver certeza se seu uso de nomes ou assets de marca SGG / SunGear Games e permitido, entre em contato com o autor do projeto ou abra uma issue primeiro.
