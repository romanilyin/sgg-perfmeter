# Política De Uso De Marca SGG / SunGear Games

Versión 1.0 - 2 de junio de 2026

Este documento describe reglas sencillas para usar los nombres y assets de marca SGG / SunGear Games.

Este repositorio está licenciado bajo [LICENSE.md](../../LICENSE.md), con el texto ruso autoritativo de la licencia en [LICENSE.ru.md](../../LICENSE.ru.md). La licencia puede permitir el uso del código, paquete u otros materiales del repositorio bajo sus términos, pero no concede permiso para usar la marca SGG / SunGear Games como nombre o branding de tus propios proyectos, productos, paquetes, forks, organizaciones, dominios, sitios web o publicaciones.

Esta política no impide referencias factuales precisas a SGG / SunGear Games por origen, compatibilidad, créditos o atribución requerida. La regla principal es sencilla: no uses la marca SGG / SunGear Games como tu propia marca y no sugieras conexión, soporte o endorsement oficial.

## Nombres Protegidos

Por favor, no uses los nombres siguientes, ni variantes que puedan confundirse con ellos, en tus propios nombres, branding, dominios, organizaciones, repositorios, paquetes, listings de marketplace o builds públicos:

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

Esto también incluye variantes similares con distinta capitalización, espaciado, guiones, puntos, guiones bajos, transliteración, escritura cirílica o errores ortográficos intencionales si pueden crear confusión con SGG / SunGear Games.

Por ejemplo, evita variantes como `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames` o `SGG-Games` cuando se usen en el contexto de software, herramientas, juegos o assets de una forma que pueda parecer una abreviatura, división, producto oficial o marca relacionada de SunGear Games.

Esta política no reclama derechos exclusivos sobre palabras genéricas o descriptivas por sí solas. Por ejemplo, el nombre `PerfMeter` sin `SGG`, `SunGear Games`, `SunGearGames` o branding similar no es objeto de esta política.

## Regla General

No nombres tus proyectos, productos, paquetes, forks, builds, organizaciones, sitios web, dominios o publicaciones de una forma que pueda hacer pensar a los usuarios que están conectados oficialmente con SGG / SunGear Games, publicados por SGG / SunGear Games, mantenidos por SGG / SunGear Games o respaldados por SGG / SunGear Games.

## No Permitido

Sin permiso separado, no uses nombres SGG / SunGear Games:

- en el nombre de tu propio proyecto, producto o paquete;
- en el nombre de una organización, repositorio o cuenta de GitHub;
- en el nombre de un paquete Unity, publicación de Asset Store u otro listing de marketplace;
- en un dominio, subdominio, sitio web o landing page;
- en un logo, icono, splash screen, branding de UI o material de marketing;
- en el nombre de una build modificada, fork o versión reempaquetada.

Ejemplos de nombres que debes evitar:

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

Esta lista no es exhaustiva. La regla sencilla es: no uses `SGG`, `SunGear Games`, `SunGearGames` ni variantes que puedan confundirse con ellos como parte de tu propia marca.

## Uso Factual Permitido

Puedes usar nombres SGG / SunGear Games para atribución factual precisa, documentación o cumplimiento de licencia.

Por ejemplo:

> Based on software by SunGear Games.

O:

> This project uses code from a SunGear Games repository.

Ese uso debe ser descriptivo. No debe parecer el nombre de tu propio producto y no debe sugerir conexión, soporte o endorsement oficial.

## Forks Regulares De GitHub

Los forks regulares de GitHub usados para pull requests, code review, aprendizaje o experimentos personales no necesitan cambiar de nombre dentro de la interfaz de forks de GitHub.

Sin embargo, si publicas tus propias releases, builds, paquetes, sitios web, documentación, listings de marketplace u otras versiones públicas, no uses SGG / SunGear Games como nombre o branding de esa versión.

## Logos Y Assets De Marca

Los logos, iconos, imágenes, screenshots con marca, branding de paquete y otros assets de marca de SGG / SunGear Games no pueden usarse en proyectos, productos, publicaciones, builds o materiales de marketing de terceros sin permiso separado.

## Preguntas

Si no tienes claro si tu uso de nombres o assets de marca SGG / SunGear Games está permitido, contacta primero con el autor del proyecto o abre un issue.
