# SGG / SunGear Games Brand Usage Policy

Version 1.0 - June 2, 2026

이 문서는 SGG / SunGear Games name 및 brand asset 사용에 관한 간단한 규칙을 설명합니다.

이 repository는 [LICENSE.md](../../LICENSE.md)에 따라 라이선스되며, 기준 러시아어 라이선스 텍스트는 [LICENSE.ru.md](../../LICENSE.ru.md)에 있습니다. license는 조건에 따라 code, package 또는 기타 repository material 사용을 허용할 수 있지만, SGG / SunGear Games brand를 자신의 project, product, package, fork, organization, domain, website, publication의 name 또는 branding으로 사용할 권한을 부여하지 않습니다.

이 policy는 origin, compatibility, credits 또는 required attribution을 위한 SGG / SunGear Games에 대한 정확한 factual reference를 막지 않습니다. 기본 규칙은 단순합니다. SGG / SunGear Games brand를 자신의 brand로 사용하지 말고, official connection, support, endorsement를 암시하지 마십시오.

자신의 names, branding, domains, organizations, repositories, packages, marketplace listings, public builds에서 다음 name 또는 혼동될 정도로 유사한 variant를 사용하지 마십시오.

예를 들어 software, tools, games, assets의 맥락에서 SunGear Games의 shorthand, division, official product 또는 related brand처럼 보일 수 있는 방식으로 `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames`, `SGG-Games` 같은 variant를 사용하지 마십시오.

이 policy는 일반적이거나 설명적인 단어 자체에 대한 독점권을 주장하지 않습니다. 예를 들어 `SGG`, `SunGear Games`, `SunGearGames` 또는 유사 branding이 없는 `PerfMeter` name은 이 policy의 대상이 아닙니다.

## 일반 규칙

사용자가 project, product, package, fork, build, organization, website, domain, publication이 SGG / SunGear Games와 공식적으로 연결되어 있거나, SGG / SunGear Games가 release, maintain, endorse한다고 생각할 수 있는 방식으로 이름을 붙이지 마십시오.

## 허용되지 않는 사용

별도 허가 없이 SGG / SunGear Games name을 다음에 사용하지 마십시오.

- 자신의 project, product 또는 package name;
- GitHub organization, repository 또는 account name;
- Unity package, Asset Store publication 또는 기타 marketplace listing name;
- domain, subdomain, website 또는 landing page;
- logo, icon, splash screen, UI branding 또는 marketing material;
- modified build, fork 또는 repackaged version name.

## 허용되는 Factual Use

정확한 factual attribution, documentation 또는 license compliance를 위해 SGG / SunGear Games name을 사용할 수 있습니다.

예:

> Based on software by SunGear Games.

또는:

> This project uses code from a SunGear Games repository.

이러한 사용은 descriptive여야 합니다. 자신의 product name처럼 보여서는 안 되며 official connection, support, endorsement를 암시해서는 안 됩니다.

## 일반 GitHub Forks

pull request, code review, learning 또는 personal experiment에 사용되는 일반 GitHub fork는 GitHub fork interface 안에서 rename할 필요가 없습니다.

그러나 자체 release, build, package, website, documentation, marketplace listing 또는 기타 public version을 publish하는 경우에는 해당 version의 name 또는 branding으로 SGG / SunGear Games를 사용하지 마십시오.

## Logos 및 Brand Assets

SGG / SunGear Games logos, icons, images, branded screenshots, package branding 및 기타 brand asset은 별도 허가 없이 third-party project, product, publication, build 또는 marketing material에서 사용할 수 없습니다.
