# Politique D'utilisation De La Marque SGG / SunGear Games

Version 1.0 - 2 juin 2026

Ce document decrit des regles simples pour l'utilisation des noms et assets de marque SGG / SunGear Games.

Ce depot est sous licence [LICENSE.md](../../LICENSE.md), avec le texte de licence russe faisant autorite dans [LICENSE.ru.md](../../LICENSE.ru.md). La licence peut autoriser l'utilisation du code, du package ou d'autres elements du depot selon ses conditions, mais elle n'accorde pas l'autorisation d'utiliser la marque SGG / SunGear Games comme nom ou branding de vos propres projets, produits, packages, forks, organisations, domaines, sites web ou publications.

Cette politique n'empeche pas les references factuelles exactes a SGG / SunGear Games pour l'origine, la compatibilite, les credits ou l'attribution requise. La regle principale est simple: n'utilisez pas la marque SGG / SunGear Games comme votre propre marque et ne suggerez pas de lien officiel, support ou approbation.

## Noms Proteges

Veuillez ne pas utiliser les noms suivants, ni des variantes pouvant preter a confusion, dans vos propres noms, branding, domaines, organisations, depots, packages, listings marketplace ou builds publics:

- `SGG`
- `S.G.G.`
- `S G G`
- `SunGear Games`
- `SunGearGames`
- `Sun Gear Games`
- `Sun-Gear Games`
- `SunGear-Games`
- `Sun-Gear-Games`
- `Sun_Gear_Games`
- `SunGear_Games`
- `Sungear Games`
- `SungearGames`
- `sungeargames`
- `sun-gear-games`
- `sun_gear_games`
- `sungear-games`
- `sungear_games`
- `sungeargames.com`
- `SunGear Games LLC`

Cela inclut aussi les variantes similaires avec capitalisation, espaces, tirets, points, underscores, translitteration, orthographe cyrillique ou fautes intentionnelles differentes si elles peuvent creer une confusion avec SGG / SunGear Games.

Par exemple, veuillez eviter des variantes comme `SunGear`, `Sun Gear`, `Sungear`, `Sun-Gear`, `SGGames`, `SGGGames` ou `SGG-Games` lorsqu'elles sont utilisees dans le contexte de logiciels, outils, jeux ou assets d'une maniere qui peut ressembler a un raccourci, une division, un produit officiel ou une marque liee a SunGear Games.

Cette politique ne revendique pas de droits exclusifs sur des mots generiques ou descriptifs par eux-memes. Par exemple, le nom `PerfMeter` sans `SGG`, `SunGear Games`, `SunGearGames` ou branding similaire n'est pas concerne par cette politique.

## Regle Generale

Ne nommez pas vos projets, produits, packages, forks, builds, organisations, sites web, domaines ou publications d'une maniere qui pourrait faire croire aux utilisateurs qu'ils sont officiellement connectes a SGG / SunGear Games, publies par SGG / SunGear Games, maintenus par SGG / SunGear Games ou approuves par SGG / SunGear Games.

## Non Autorise

Sans autorisation separee, n'utilisez pas les noms SGG / SunGear Games:

- dans le nom de votre propre projet, produit ou package;
- dans le nom d'une organisation, d'un depot ou d'un compte GitHub;
- dans le nom d'un package Unity, d'une publication Asset Store ou d'un autre listing marketplace;
- dans un domaine, sous-domaine, site web ou landing page;
- dans un logo, une icone, un splash screen, un branding UI ou un materiel marketing;
- dans le nom d'un build modifie, fork ou version repackagee.

Exemples de noms a eviter:

- `SGG Tools`
- `SGG Unity Tools`
- `SGG PerfMeter Fork`
- `SGG PerfMeter Pro`
- `SGG Performance Toolkit`
- `SunGear Games Toolkit`
- `SunGear Unity`
- `SunGear Tools`
- `SunGearGames Extensions`
- `SunGearGames Community Edition`
- `sungeargames-tools`
- `sungear-perfmeter`
- `sgg-perfmeter-pro`
- `sun-gear-unity-tools`
- `sgg-unity-assets`

Cette liste n'est pas exhaustive. La regle simple est: n'utilisez pas `SGG`, `SunGear Games`, `SunGearGames` ou des variantes pouvant preter a confusion comme partie de votre propre marque.

## Usage Factuel Autorise

Vous pouvez utiliser les noms SGG / SunGear Games pour une attribution factuelle exacte, de la documentation ou la conformite a la licence.

Par exemple:

> Based on software by SunGear Games.

Ou:

> This project uses code from a SunGear Games repository.

Cet usage doit etre descriptif. Il ne doit pas ressembler au nom de votre propre produit et ne doit pas suggerer de lien officiel, support ou approbation.

## Forks GitHub Ordinaires

Les forks GitHub ordinaires utilises pour les pull requests, revues de code, apprentissage ou experimentations personnelles n'ont pas besoin d'etre renommes dans l'interface de fork de GitHub.

Cependant, si vous publiez vos propres releases, builds, packages, sites web, documentation, listings marketplace ou autres versions publiques, n'utilisez pas SGG / SunGear Games comme nom ou branding de cette version.

## Logos Et Assets De Marque

Les logos, icones, images, captures d'ecran marquees, branding de package et autres assets de marque SGG / SunGear Games ne peuvent pas etre utilises dans des projets, produits, publications, builds ou materiels marketing tiers sans autorisation separee.

## Questions

Si vous n'etes pas sur que votre utilisation des noms ou assets de marque SGG / SunGear Games soit autorisee, veuillez contacter l'auteur du projet ou ouvrir une issue d'abord.
