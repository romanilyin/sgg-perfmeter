# Release Candidate 2026.5.20-1

Статус: private/internal release candidate. Repository stays private until an explicit public switch is approved.

## Scope

- Runtime API `SGG.PerfMeter.PerformanceMeter` for lifecycle, status, metrics, overlay controls, collection modes, target FPS, sessions, alerts, custom metrics, device/camera snapshots, Render Graph snapshots, overdraw measurement and heatmap visibility.
- Project-owned JSON settings for zero-code setup, overlay presets/modules/tunables, rule defaults, session defaults and overdraw limits.
- UI Toolkit runtime overlay with allocation-conscious text refresh and bounded custom metric rows.
- URP 17 Render Graph feature with opt-in overlay marker, numerical overdraw pass, visual heatmap pass and safe Render Graph analytics snapshot.
- Editor setup/runtime/presets/debug window and MCP command handlers.
- Package Manager samples for bootstrap/settings, runtime workflows and editor/MCP automation.
- EditMode and PlayMode test suites.
- Android smoke-build helper and Android runtime smoke bootstrap.

## Known Limits

- Current render integration is URP-only.
- Built-in Render Pipeline is unsupported.
- HDRP support is planned, not implemented.
- GPU timings can be unavailable or unreliable depending on platform and graphics API.
- OpenGL/OpenGLES is degraded for numerical overdraw instrumentation.
- Overdraw measurement and heatmap add diagnostic redraw cost while active.
- Full self-overhead subtraction is not finalized.
- Render Graph internal counters can degrade to `-1` when Unity does not expose them safely.
