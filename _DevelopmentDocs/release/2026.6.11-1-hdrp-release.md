# 2026.6.11-1 HDRP Release

Статус: release pass for HDRP support.

## Scope

- Version/tag: `2026.6.11-1`.
- Package: `com.sungeargames.perfmeter`.
- Runtime target: Unity `6000.4+` with URP `17.4+` Render Graph or HDRP `17.4+` Custom Pass integration.
- Built-in Render Pipeline remains unsupported.
- HDRP overdraw and heatmap remain unsupported; core diagnostics remain available.

## Release Notes

- Added SRP detection without hard dependencies on URP or HDRP.
- Added HDRP camera snapshots through reflection over `HDAdditionalCameraData`.
- Added optional HDRP assembly and runtime-registered HDRP Custom Pass integration.
- Extended render integration snapshots and MCP output with pipeline/integration/injection-point metadata.
- Updated setup/status flows for HDRP and skipped URP renderer mutation in HDRP projects.
- Added explicit unsupported state for HDRP overdraw and heatmap.
