## Currently Supported In First Public Release

| Area | Supported now |
| --- | --- |
| Unity version | Official target: Unity `6000.4+`. Older Unity versions are import-safe only if package metadata still allows import; they are not supported targets. |
| Render pipeline | URP `17.4+` with Render Graph integration. |
| Built-in Render Pipeline | Not supported and not planned. |
| HDRP | Planned future work, not implemented in `2026.6.5-1`. |
| Runtime metrics | FPS/lows/spikes, CPU frame/main/render/present wait timings, GPU timing when available, render counters, SRP Batcher, BRG/GRD, index upload, memory and GPU memory counters where Unity exposes them. |
| Bottleneck classification | GPU-bound, CPU main-thread-bound, CPU render-thread-bound, present-limited, balanced and unknown states. |
| Runtime overlay | UI Toolkit overlay with presets/modules, graph/text/full display modes, target FPS line, throttled allocation-conscious text refresh and bounded custom metric rows. |
| URP render integration | `PerfMeterRenderGraphFeature` for markers, numerical overdraw measurement, visual overdraw heatmap and safe Render Graph observation. |
| Overdraw | Opt-in numerical overdraw measurement and visual heatmap for supported graphics backends; OpenGL/OpenGLES are degraded/unsupported for numerical instrumentation. |
| Sessions | Bounded session recorder, warm-up, scene scope, worst-frame summaries and JSON/CSV export. |
| Alerts | Rule alerts with structured logs, callbacks, Editor warning cooldowns and MCP alert commands. |
| Custom metrics | Public provider API, latest metrics output, session export and limited overlay rows. |
| Device/camera metadata | Device/environment snapshots and camera snapshots with URP camera settings when available. |
| Automation | Public runtime API and MCP commands for setup/status/metrics/device/camera/render graph/overlay/overdraw/sessions/alerts. |
| Editor tooling | Setup/runtime/presets/debug window for Frame Timing Stats, URP renderer feature setup, JSON settings, runtime controls and diagnostics. |
| Samples | Package Manager samples for bootstrap/settings, runtime workflows and editor/MCP automation. |
| Android validation | Current smoke validation covers Galaxy S23 Vulkan and OpenGLES3 fallback; broader device coverage is not claimed. |

## Current Known Limits To Mention Publicly

- GPU timings can be delayed, unavailable or unreliable depending on platform and graphics API.
- Some `ProfilerRecorder` counters can be unavailable on specific platforms or Unity versions.
- Render Graph internal counters are conservative and can degrade to `-1` when Unity does not expose them safely.
- Overdraw measurement and heatmap are diagnostic modes and add extra rendering cost while active.
- Full self-overhead subtraction is not finalized.
- This release complements Unity Profiler, Frame Debugger, RenderDoc and Profile Analyzer; it does not replace deep frame capture tools.
