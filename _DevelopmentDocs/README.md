# SGG PerfMeter Development Docs

Этот каталог предназначен только для внутренней разработки пакета. Пользовательская документация живет в `docs/en`, `docs/ru` и package-local `.Documentation`.

## Структура

| Раздел | Назначение |
| --- | --- |
| `backlog/` | Полезные, но еще не реализованные задачи и направления. |
| `decisions/` | Короткие технические решения: что поддерживаем, что не поддерживаем и почему. |
| `release/` | Минимальные внутренние материалы для подготовки private release candidate. |

## Правила

- Не хранить здесь пользовательские guides или публичные продуктовые тексты.
- Не хранить историю уже завершенных итераций, если результат вошел в release notes или changelog.
- Backlog должен описывать только будущую работу и текущие ограничения.
- Release-доки должны быть короткими и актуальными для текущего private release candidate.
