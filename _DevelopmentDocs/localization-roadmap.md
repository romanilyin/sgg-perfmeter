# Localization Roadmap

The current user-facing documentation is maintained in English and Russian:

- `docs/en/`
- `docs/ru/`

Planned documentation localization targets:

- `.de`
- `.es`
- `.fr`
- `.it`
- `.ja`
- `.ko`
- `.pt-BR`
- `.ru`
- `.zh-CN`

Public README files should show only currently available languages. Future language targets belong in this development roadmap until translations are ready.
