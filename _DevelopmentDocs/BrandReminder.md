# Brand Usage Reminder Template

Use this text as a friendly reminder when another repository, package, listing, build, or fork appears to use SGG / SunGear Games branding in a confusing way.

```md
Brand usage reminder

Hi.

Please note that this repository has a brand usage policy:

- docs/en/brand.md
- docs/ru/brand.md

The repository license allows use of the code/package under its terms, but it does not grant permission to use the SGG / SunGear Games name or branding for separate projects, packages, forks, builds, domains, organizations, or marketplace listings.

Please do not use names such as `SGG`, `SunGear Games`, `SunGearGames`, or confusingly similar variants as part of your own project name or branding.

You can still mention the origin factually, for example:

"Based on software by SunGear Games"

This is a friendly reminder to avoid possible user confusion and branding-related issues later on.

Thanks.
```
